# vbase-admin-template

> VUE 后台管理系统架构模板

## Build Setup

```bash
# 克隆项目
git clone https://github.com/amilytom/vbase-admin-template.git

# 安装依赖
npm install

# 启动服务 http://localhost:8080
npm run dev

# 发布项目
npm run build
```

## 功能

- 登录/注销
- 权限验证（页面级）
- 动态侧边栏
- 动态面包屑
- 导航标签
- 401、404、全局错误捕捉
- element-ui
- axios 封装（统一处理请求、拦截、报错等）
- cache 封装
